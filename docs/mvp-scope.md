# MVP Scope

## Mobile App

- User registration and login.
- Home screen with `Report Incident` and `Call 112`.
- Offline-first report creation.
- GPS capture and GhanaPostGPS fallback field.
- Image evidence upload.
- Voice note recording, playback, delete, and re-record.
- Background synchronisation to Firebase.
- My Reports and report status tracking.

## Admin Dashboard

- Admin login.
- Overview metrics.
- Report list and filters.
- Report detail view with evidence, voice note, and location.
- Report status updates, internal notes, and spam flag.
- Real-time urgent report alert.
- User list.

## Firebase Backend

- Firebase Authentication.
- Cloud Firestore.
- Firebase Storage.
- Cloud Functions.
- Firestore and Storage security rules.
- Custom claims for role-based admin access.
- Firebase Emulator workflow for local development.
